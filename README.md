## üì¶ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/akhildev-mj/portfolio.git
cd portfolio
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Start the Dev Server

```bash
pnpm run dev
```

Open `http://localhost:3000` in your browser.
